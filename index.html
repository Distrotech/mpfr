(the last argument can be any directory name). You can use

svn ls svn://scm.gforge.inria.fr/svn/mpfr/branches svn ls svn://scm.gforge.inria.fr/svn/mpfr/tags

to get the list of branches or tags (releases), then checkout a particular branch or tag instead of the trunk. Alternatively, you can now use the "https:" scheme (a.k.a. DAV) instead of "svn:". For more information about Subversion, please see:
